# qiita-adv-calendar

Display items with many like items in order in the [qiita advent calendar](https://qiita.com/advent-calendar/2017)

## how to use

```bash
$ git clone git@github.com:smith-30/qiita-adv-calendar.git
$ mv .env.sample .env
# edit .env
$ dep ensure -v
$ go run main.go
```
